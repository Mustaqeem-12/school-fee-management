# School Fee Management System

This is a web application to manage school student fee payments.

## Features
- Student registration
- Fee payment tracking
- Payment history
- Admin dashboard
- Secure login

## Technologies Used
- ASP.NET
- C#
- SQL Server
- HTML, CSS, JavaScript

## How to Run
1. Clone the repository
2. Open project in Visual Studio
3. Run the project
